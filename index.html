# L4D2bot / 求生机器人

欢迎来到 L4D2bot / 求生机器人 项目！这是一个求生之路服务器的管理项目，旨在帮助您轻松、高效地管理您的求生服务器。

需要说明的是，当前并没有独立的bot，而是依附于nonebot2的插件形式，名为nonebot_plugin_l4d2_server。

# 开始

要开始使用 L4D2bot，请访问我们的 文档，了解如何安装、配置和使用此项目。

本文档尚未晚上，建议先移步这里 (opens new window)查看文档

# 功能

  • qq上传求生地图
  • 快速查询指定服务器状态
  • 批量一键查询多服务器状态
  • 创意工坊下载
  • 求生喷漆制作
