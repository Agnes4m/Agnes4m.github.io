          # 安装 Python 3

          插件支持python3.8+

          # windows

          1、前往 Python 官网下载 Python 安装包：https://www.python.org/downloads/windows/

          2、选择合适的版本，推荐下载 Python 3.8 - 3.10 的任意版本

          3、下载安装包后，双击打开安装包进行安装

          4、按照提示进行安装，推荐将 Python 添加到环境变量path中，同时安装pip指令

          5、在cmd命令行中使用指令python，没有报错即可正常使用

          # linux

          1、Ubuntu 20.04（py3.8.10) / Ubuntu 22.04(py3.10.6)系统：

          sudo apt-get update
sudo apt-get install python3
sudo apt-get install python3-pip

          2、其他系统，下载官网对于版本：https://www.python.org/downloads/source/

          # 安装 Nonebot 2

          在命令行中运行以下命令来安装 Nonebot 2：

          pip install nonebot2
pip install nb-cli

          # 安装-gocq

          方法一：

          使用内置插件nonebot-plugin-gocqhttp

          方法二：

          点击这里 (opens new window)下载最新版本对应的gocq，对应系统启动一次程序，再修改config文件，就可以正常启动了

          # 安装求生插件

          以下方法任选其一即可

          nb plugin install nonebot_plugin_l4d2_server

          pip install nonebot_plugin_l4d2_server
